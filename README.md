# target-case-study

#### Prerequisite

Node v10.x.x

### Set up
```sh
$ git clone https://github.com/sekhar-ashwin/target-case-study.git
$ cd target-case-study
$ npm install
```

####  Run the server
```sh
$ npm start
```
###  API paths
```sh
Get product details
Method: GET
http://localhost:3000/products/:id

Update product price
Method: POST/PUT
http://localhost:3000/products/:id
```

###  Run unit test
```sh
$ npm test
```
